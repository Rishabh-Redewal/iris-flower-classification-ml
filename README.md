# Iris Flower Classification ML

A Machine Learning project built using Scikit-learn to classify Iris flower species based on flower measurements.

## Features
- Iris flower classification using Machine Learning
- Model training on Iris dataset
- Accuracy evaluation of predictions
- Data preprocessing using Scikit-learn

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Project Files
- accuracy_on_iris.ipynb → Model training and accuracy evaluation
- quick_training_on_iris.ipynb → Initial model training
- iris-Train → Training dataset
- iris-Test → Testing dataset
